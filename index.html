Helix Font for QMK

by @wavebeem

This tool can be used embed two images within your Helix QMK keyboard font. It also contains code snippets on how to display those images.

Upload Images

Images should be 32px wide by 96px tall in black and white. Actual colors on your OLED screen may vary.

Preview Font

Font Code

Save this code as helixfont.h or glcdfont.c depending on your QMK setup.

Render Image 0 Code

Use this code to print image 0 to your OLED display.

matrix_write_P(matrix, PSTR(
